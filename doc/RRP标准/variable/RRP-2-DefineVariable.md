# RRP标准-2-定义变量

## 变量的定义
在Vanction语言中，变量的定义如下：
```Vanction
int variable_name = value; || 定义一个整型变量variable_name并赋值为value
string variable_name = value; || 定义一个字符串变量variable_name并赋值为value
List variable_name = value; || 定义一个列表变量variable_name并赋值为value
HashMap variable_name = value; || 定义一个哈希表变量variable_name并赋值为value
auto variable_name = value; || 定义一个变量variable_name，值为value
define variable_name; || 定义一个变量variable_name，值为undefined
```

## 使用变量
在Vanction语言中，使用变量的格式如下：
```Vanction
System.print(variable_name); || 使用变量variable_name
```
